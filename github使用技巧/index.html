发表于 2020-03-24 | 分类于 实践案例 |

搜索技巧

  1. 指定搜索范围
    in:name xxx // 按照项目名搜索
    in:readme xxx // 按照README搜索
    in:description xxx // 按照description搜索

  2. 增加筛选条件：
    stars:>xxx // stars数大于xxx
    forks:>xxx // forks数大于xxx
    language:xxx // 编程语言是xxx
    pushed:>YYYY-MM-DD // 最后更新时间大于YYYY-MM-DD

  3. 模式：in:[搜索范围] 关键字 [搜索条件]:[条件]

代码无法推送实例与解决方案

  1. ping github.com 看看能否访问网址。如果不行，在/etc/hosts中设置映射

  2. 检查代理，如果代理没有问题，则使用流量而非公司wifi

  3. 302

  4. 405

  5. 32 unable to access
