line-height: normal的高度是如何计算的？

<p> <span class="a">Ba</span> <span class="b">Ba</span> </p>

  1. 背景：font-size: 100px 在不同的 font-family 下的<p>元素的 height 不同
  2. line-box(每行文本的虚拟行)的高度由line-heigh和 vertical-align公共决定
  3. 对于p元素来说：其每行 line-box 都隐含一个没有宽度的内容区域元素（font-size, font-family继承自p元素）。其与其它内容元素通过vertical-align对齐之后的最大高度即为 p元素的line-height: normal的高度

line-height 如果指定为数字则相对于p元素的font-size大小

其它定义：

  1. line-box 行盒：段落的每一行每一行都视为一个行盒

  2. content-area 内容区域：每个元素font-size, font-family确定之后字体渲染的高度（含基线以上的Ascent，基线以下的Decent）

参考：

  1. https://www.w3cplus.com/css/css-font-metrics-line-height-and-vertical-align.html

类flex布局

  1. 使用 float:left/right 也能使得其它兄弟元素自动扩展宽度，类似于 flex:1的效果
