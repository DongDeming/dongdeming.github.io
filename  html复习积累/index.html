html复习积累

发表于 2023-04-23 | 分类于 前端开发 |

<base>

  • 所有的css/html中的相对路径，锚点都会使用base作为基准
  • 锚点<a href="#part2">part2</a>一般常用于页内定位。使用<base>会导致因为是相对路径生成新的URL触发请求新页面，从而导致页面内定位的功能失效，此时需要使用绝对定位来做锚点。

<img>

  • 具备懒加载、预加载、加载优先级等属性设置

form property

  • <button form="form1id">submit</button> 能够关联对应form 元素，并且不受嵌套影响。

viewport

  • layout viewport: 实际的document渲染的视图
  • visual viewport:
    • 当前可见的部分layout vieport大小。
    • 不同于window.screen是固定大小的，其会受到手势缩放影响。缩小时，visual viewport 可见的layout viewport 部分变多，反之变少。
    • 可通过 控visual-viewport 宽度与设备的逻辑像素一致。如当不设置 时，屏幕逻辑像素是375, visual viewport 是980， 而layout viewport 则根据其渲染宽度决定。html {width: 100%} 即根据window.innerWidth 约等于visual viewport
